Exercício para criação de uma página HTML com as seguintes configurações.



  • Exiba com um alerta de mensagem de boas vindas ao usuário.
    • Bem-vindo! A seguir pediremos que informa alguns dados.
  • Peça que o usuário informe seu nome.
  • Peça que o usuário informe sua idade
  • Peça que ele confirme a idade
  • Exiba um alerta:
    • O nome do usuário digitado.
    • A idade digitada.
    • A confirmação da idade.
