While Estrutura de repetição

  • Ele executa um laço de repetição, parecido com if
  • Ele fica repetindo enquanto determinada condição for verdadeira
  • O break para a repetição em qualquer momento.
  • Ele precisa de uma condição falsa

Sintaxe:

let velocidade = 80
while(velocidade > 0){
alert("O carro está a " + velocidade + "km/h.")
velocidade -= 20
alert("Diminuindo 20 km/h")
}
alert("O carro parou") // nesse caso seria a alternativa "false" pq o carro está 0 km, nesse caso para o While.
